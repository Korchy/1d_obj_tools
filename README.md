# 1D OBJ Tools

Blender add-on.

Add-on functionality
-
Some tools for working with OBJ files.

### Shift

Shifts all vertices by X,Y,Z coordinates on specified value.

Creates modified OBJ file with "_s" postfix in the name in the same directory.
